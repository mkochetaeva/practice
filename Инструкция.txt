1. Перед запуском убедитесь, что у вас установлен:
Python 3.8 или новее
pip
2. Скачайте или скопируйте содержимое файла requirements.txt и выполните
 в командной строке/терминале: pip install -r requirements.txt
3. Сохраните основной код приложения в файл ImageApp.py и запустите приложение
 командой: python ImageApp.py
 Так же для запуска можно использовать исполняемый файл ImageApp
